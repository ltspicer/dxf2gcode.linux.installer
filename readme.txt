Just start in the console with:
./linux.installer.sh
or double-click and select "run in terminal".
Then follow the instructions. The script installs all necessary packages and dxf2gcode automatically.

For support: https://www.ltspiceusers.ch/#dxf2gcode.68

Python 3.7 or higher is required.
Tested on Linux Mint 20.3 mate and Lubuntu 20.04.4
